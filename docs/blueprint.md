# **App Name**: KabaWeb

## Core Features:

- Multilingual Content: Display content in both French and English based on user preference, fetched from Firestore.
- News Publication: Enable admins to publish news articles with titles, content, and publication status via Firestore.
- Contact Form: Allow users to submit messages via a contact form, which are then stored in Firestore.
- Admin Panel: Provide a secure admin panel for managing multilingual content, news articles, and user messages. Access managed with Firebase Authentication.
- Secure Authentication: Implement Firebase Authentication to secure the admin panel, ensuring only authorized users can manage content.
- SEO Optimization: Uses the OpenAI tool to suggest a new page title and meta description, to improve the SEO results for that page.

## Style Guidelines:

- Primary color: Rich, deep red (#B91C1C) to evoke trust and a sense of importance.
- Background color: Soft, muted gray (#F9FAFA) to provide a clean and modern backdrop.
- Accent color: Golden yellow (#A16207) as a highlight color that will provide contrast.
- Headline font: 'Playfair', a serif font, for titles and important text; its modern style and high contrast give it an elegant, high-end feel.
- Body font: 'PT Sans', a sans-serif font, for body text; a classic sans-serif that pairs well with Playfair.
- Use professional, minimalist icons for key functions and navigation. Ensure icons are easily recognizable and consistent in style.
- Implement a clean, responsive layout that adapts seamlessly to different screen sizes. Use clear visual hierarchy to guide users through content.
- Incorporate subtle transitions and animations to enhance user experience, such as smooth scrolling and loading indicators.
